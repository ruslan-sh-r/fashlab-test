**FashLab invoice uploader**
****

How to deploy *backend*:
* Install mysql
* Go to back/config/local.json and set db connect options
* run command in console: "npm install"
* run command in console: "npm run start:watch"

How to deploy *frontend*:
* go to front folder
* run command in console: "npm install"
* run command in console: "ng serve"
* navigate to http://localhost:4200/ in browser

How to run backend tests:
* go to back folder
* run command in console: "npm run test"
